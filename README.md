# Stord

Stord is a commerce fulfillment platform that combines physical warehouse infrastructure with integrated software to manage omnichannel logistics for DTC and B2B brands. The platform provides REST APIs for managing orders, inventory, shipments, and returns across a network of 20+ fulfillment nodes, with pre-built integrations for Shopify, BigCommerce, Magento, WooCommerce, Amazon, and ERP systems such as Oracle NetSuite.
